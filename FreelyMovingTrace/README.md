# 旷场实验行为学分析项目

**Open Field Behavioral Tracking & Analysis**

## 📂 项目概览

本文件夹包含旷场实验的**原始视频数据**、**MATLAB 追踪算法脚本**以及**批量处理后的结果汇总**。
主要用于提取小鼠的运动轨迹（X, Y 坐标），计算运动距离、速度，并生成可视化的轨迹图。

---

## 🛠️ 1. 核心分析脚本 (Analysis Scripts)

* **`MouseTracker_OpenField.m`** (核心主程序)
* **功能**：这是行为学分析的主引擎。
* **流程**：读取 `.avi` 视频 -> 识别黑色小鼠（背景减除法/阈值法） -> 记录每帧坐标 -> 生成 `.csv` 数据表和轨迹图。
* **输出**：它会为每个视频创建一个独立的 `_Results` 文件夹。


* **`CSV_Trajectory_Cleaner.m`** (数据清洗工具)
* **功能**：用于修复追踪质量不佳的数据。
* **用途**：如果原始追踪结果中有“瞬移”（跳变）或严重的锯齿抖动，运行此脚本可以平滑轨迹，剔除异常点，并重新导出干净的 `.fig` 和 `.csv`。

---

## 🚀 建议处理流程 (Workflow)

1. **准备数据**：将所有 `.avi` 视频放在此文件夹中。
2. **运行追踪**：
* 打开 MATLAB，运行 `MouseTracker_OpenField.m`。
* 选择所有视频文件。
* 程序会自动处理，并在最后生成 `All_Trajectories_Summary`。


3. **检查质量**：
* 打开生成的图片，检查轨迹是否连贯。
* 如果有断裂或乱跳，使用 `CSV_Trajectory_Cleaner.m` 对特定数据进行修复。


4. **数据统计**：
* 打开各个 `_Results` 文件夹里的 `.csv` 文件，获取具体的距离和速度数据进行统计学分析。
